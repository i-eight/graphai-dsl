# Date

## now

```
now = () -> number
```

### Return value

The current time in milliseconds since the Unix epoch.

### Example

```
Date.now()
```

## toISOString

```
toISOString = (date: Date) -> string
```

### Parameters

| Name | Description |
| :--- | :--- |
| date | A Date object. |

### Return value

A string representing the date in the simplified extended ISO format.

### Example

```
Date.toISOString(Date.fromISOString('2025-03-01T15:08:36.533Z')) == '2025-03-01T15:08:36.533Z';
```

## fromISOString


### Parameters

| Name | Description |
| :--- | :--- |

### Return value

### Example

```

```

## toEpochTime

```
toEpochTime = (date: Date) -> number
```

### Parameters

| Name | Description |
| :--- | :--- |
| date | A Date object. |

### Return value

A number representing the number of milliseconds since the Unix epoch.

### Example

```
Date.toEpochTime(Date.fromEpochTime(1740841716533)) == 1740841716533;
```
