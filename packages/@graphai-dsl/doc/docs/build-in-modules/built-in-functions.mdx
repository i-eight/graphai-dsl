# Built-in Functions

## println

```
println = (msg: unknown) -> void
```

### Parameters

| Name | Description|
| :--- | :--- |
| msg | A message to print  |

### Return value

void

### Example

```
println('Hello World');
```

## identity

```
identity = <A>(a: A) -> A;
```

### Parameters

| Name | Description|
| :--- | :--- |
| a | Any value to return |

### Return value

The same value as the input.

### Example

```
identity(1) == 1;
```

## loop

```
loop = <A>(init: A, f: (a: A) -> A) -> A;
```

### Parameters

| Name | Description|
| :--- | :--- |
| init | An initial value |
| f | A callback function. |

### Return value

The value that results from running the callback function to completion.

### Example

```
loop(0, (x) -> if x < 10 then recur(x + 1) else x) == 10;
```

## eval

```
eval = (code: string) -> unknown;
```

### Parameters

| Name | Description|
| :--- | :--- |
| code | A code of the GraphAI DSL |

### Return value

The value that results from running the code.

### Example

```
@version("0.6");
eval('
  @version("0.6"); 
  a = 1; 
  b = 1;
  a + b;
') == 2;
```
