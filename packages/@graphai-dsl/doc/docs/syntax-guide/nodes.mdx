# Nodes

## Computed Node

```
foo = 1;
```

```
foo = 'bar';
```

```
foo = 1 + 2;
```

```
foo = barAgent('baz');
```

```
foo = @console({after: true}) barAgent('baz');
```

## Anonymous Node

```
x = 1;
y = 2;
x + y;
```

```
x = 1;
y = 2;
println(x + y);
```

## Static Node

```
static foo = 1;
```

```
static foo = 'bar';
```
