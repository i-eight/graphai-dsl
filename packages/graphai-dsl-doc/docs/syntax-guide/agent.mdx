# Agent

## Defining a Agent

```
@version('0.6');

public sayHelloAgent = {
    name: 'sayHelloAgent',
    agent: (name) -> 'Hello, ${name}',

    description: 'This agent says hello to the given name.',

    input: {
        type: 'string',
        description: 'The name to say hello to.',
    },

    output: {
      type: 'string',
      description: 'The greeting message.',
    }
};
```

## Calling an Agent

```
sayHelloAgent('Alice') |> println;
```
