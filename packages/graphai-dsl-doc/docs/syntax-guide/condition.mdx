# Connections

```
if x > 1 then 2 else 3;
```

```
if x > 1 then {
  println("x is greater than 1");
} else {
  println("x is less than or equal to 1");
}
```

```
if x > 1 then {
  println("x is greater than 1");
} else if x == 1 then {
  println("x is equal to 1");
} else {
  println("x is less than 1");
}
```
