# Nested Graph

```
foo = {
  x = 1;
  y = 2;
  x + y;
};
```

```
println({
    x = 1;
    y = 2;
    x + y;
})
```

```
f = () -> {
    x = 1;
    y = 2;
    x + y;
}
```

```
if x > 1 then {
  println("x is greater than 1");
} else {
  println("x is less than or equal to 1");
}
```
